Long-term Underwater Object Tracking Dataset

A Benchmark for Real-World Underwater Scenarios (LTUOT)

The first long-term object tracking benchmark focusing on real underwater scenarios, integrating multi-modal dynamic language descriptions.

Key Highlights

First Underwater Long-term Benchmark

Fills the gap in the current visual object tracking field for underwater long-term scenarios. Specifically designed to solve tracking challenges such as frequent target disappearance, reappearance, and severe appearance changes in underwater environments.

100% Real-world Video Sequences

Completely abandons synthetic or semi-synthetic data. Compared to previous datasets, all video sequences here are collected from open waters like real oceans and lakes, encompassing true underwater turbidity, light scattering, and occlusions.

Global + Dynamic Dual Annotation

Not only provides a global state annotation describing the target's overall attributes, but also introduces an innovative dynamic vision-language annotation, accurately describing the target's latest state every 4 seconds.

Innovative Multi-modal Annotation System

To promote the application of Vision-Language models in object tracking, we designed a dual-layer language annotation mechanism:

  • 1

    Global Language Annotation

    Provides a global description of the target's category, initial appearance, and main colors, which remain relatively stable over time.

    "A sea turtle with yellow stripes swimming near a coral reef."
  • 2

    Dynamic Vision-Language Annotation

    Captures drastic state changes during long-term tracking. A dynamic refresh is performed every 4 seconds, detailing the target's current action, deformation, relative position, and occlusion.
